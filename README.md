## Overview

PromiseLane is a project governance platform that brings clarity 
to client-service provider relationships. It creates a single 
source of truth connecting requirements, deliverables, and 
payments — reducing disputes and building trust.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Laravel 12 (PHP 8.2+) |
| Database | MySQL 8.0 |
| Auth | JWT (tymon/jwt-auth) — stateless |
| API | RESTful, versioned (`/api/v1/`) |
| Architecture | Thin Controllers → Service Layer → API Resources |
| Frontend | React + Vite — separate repository |

---

## Architecture Principles

- Stateless JWT authentication
- Role-based access control
- Service layer for business logic
- API Resources for response transformation
- Immutable activity logging
